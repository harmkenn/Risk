# Risk Game as a Streamlit App

This project implements the classic board game Risk as a web application using Streamlit.

## Setup

1. Install dependencies:
   ```bash
   pip install streamlit
   ```

2. Run the app:
   ```bash
   streamlit run app.py
   ```

## Features
- Multiplayer support
- Interactive board
- Game state management
